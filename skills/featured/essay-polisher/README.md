Essay Polisher Skill
========

This AI Edge Gallery JS skill polishes study-abroad essays (PS/SOP/personal statement),
removes common AI-flavored wording, and rewrites weak openings/closings into
more specific and human-sounding prose.

It includes:
- `SKILL.md` for trigger + instruction logic
- `scripts/index.html` for `run_js` execution (`ai_edge_gallery_get_result`)

Typical prompts:
- "Polish my personal statement"
- "Remove AI tone from this SOP"
- "This essay sounds robotic, rewrite it naturally"
