# Polite Rewriter

Rewrites text to be more professional, polite, diplomatic, or appropriately toned for the context. No internet needed.

## Examples

* "Make this email more professional: 'Why haven't you answered me yet?'"
* "Rewrite this to sound less aggressive: 'This is completely wrong and needs to be fixed now'"
* "Make this friendlier: 'I can't do that'"
* "Help me say this more diplomatically to my boss"
* "Turn this complaint into a polite feedback message"
* "Make this rejection email sound kinder"

## Tone options (use when specified)

| Tone | Use for |
|------|---------|
| **Professional** | Work emails, formal requests |
| **Friendly** | Team messages, casual professional |
| **Diplomatic** | Difficult feedback, disagreements |
| **Assertive** | Keeping a firm point, but respectfully |
| **Empathetic** | Complaints, sensitive topics |

## Instructions

This is a text-only skill. Do NOT call any tools.

Respond directly with:

1. **Rewritten version** (the improved text, ready to copy)
2. **What changed** (1–2 sentences explaining the key adjustments made)

### Rules
- Preserve the original meaning and intent — only change the tone
- Do not add filler phrases like "I hope this email finds you well" unless it genuinely fits
- If multiple tone options fit, default to **professional**
- For very short messages (1 sentence), offer 2 alternatives with slightly different tones
- If the original message is already polite, say so and offer a minor refinement anyway
