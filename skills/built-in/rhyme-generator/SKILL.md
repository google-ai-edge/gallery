# Rhyme Generator

Helps with creative writing that rhymes — poems, song lyrics, birthday verses, raps, and more. No internet needed.

## Examples

* "Write a short poem about the ocean"
* "Help me write a birthday poem for my friend Sarah who loves cats"
* "Give me rhymes for the word 'moon'"
* "Write a funny 4-line poem about Mondays"
* "I need a chorus for a song about summer — make it rhyme"
* "Write a short rap about coffee"

## Instructions

This is a text-only skill. Do NOT call any tools.

Respond directly based on what the user needs:

### If asked for rhyming words
List 8–12 words that rhyme with the given word, grouped by rhyme quality (perfect rhymes first, then near-rhymes).

### If asked for a poem or verse
Write the poem directly. Follow these defaults unless specified:
- **Length**: 2–4 stanzas (or 8–16 lines for lyrics)
- **Rhyme scheme**: ABAB or AABB (whichever fits the tone)
- **Tone**: match the subject — playful for funny topics, heartfelt for emotional ones

### If asked for song lyrics
Structure: verse → chorus → verse (keep it singable and rhythmic)

### Rules
- Always prioritize rhythm and flow — read it aloud mentally
- Avoid forced rhymes that sound unnatural
- For personalized poems, weave in the details provided (names, hobbies, memories)
- If the topic is sad or emotional, be genuine, not cliché
