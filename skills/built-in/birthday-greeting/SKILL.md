# Birthday Greeting

Creates warm, personalized birthday messages for any person and relationship. No internet needed.

## Examples

* "Write a birthday message for my best friend Emma"
* "Help me write something for my dad's 60th birthday card"
* "Give me a funny birthday message for my colleague"
* "Write a heartfelt birthday speech for my sister's 30th"
* "What should I write in a birthday card for my grandmother?"
* "Write a birthday wish for a 5-year-old"

## Instructions

This is a text-only skill. Do NOT call any tools.

Respond directly with a personalized birthday message. Use any details provided (name, relationship, age, shared memories, personality, hobbies) to make it feel genuine.

### Tone guide

| Relationship | Default tone |
|--------------|-------------|
| Close friend | Warm, playful, personal |
| Parent / grandparent | Heartfelt, respectful |
| Partner / spouse | Romantic and personal |
| Colleague | Friendly but professional |
| Child | Fun, simple, enthusiastic |
| Boss / formal | Polite, brief |

### Format

- **Short card message**: 2–4 sentences (default for "card" requests)
- **Long message / speech**: 3–5 paragraphs with an opening, personal reflection, and warm closing
- **Funny version**: include a light joke or witty observation tied to the person or milestone age

### Rules
- Always use the person's name if provided
- For milestone ages (30, 40, 50, 60, ...) acknowledge the milestone warmly — not as "getting old"
- If no details are given, write a universally warm message and offer to personalize it further
- Avoid clichés like "hope your day is as special as you are" unless asked for something simple
